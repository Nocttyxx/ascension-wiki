Installation

Version documentée : v1.0.0, fondée sur la configuration validée RC4.

Cette page garantit que le joueur lance exactement le bon pack, sans conflit de versions ni surprises côté serveur.

ÉlémentValeurConseil
Minecraft1.20.1Ne change pas de version.
LoaderForge 47.4.0Doit être identique côté client et serveur.
Mise à jourProfil completNe pas mettre à jour les mods un par un.
SauvegardeRecommandéeÀ faire avant chaque mise à jour importante.

Étape 1

Installe le profil officiel du modpack depuis ton lanceur compatible CurseForge.

Étape 2

Lance une première fois le jeu sans rien modifier, pour générer tous les fichiers proprement.

Étape 3

Active ensuite le resource pack et le shader recommandés par le pack.

Important : la majorité des crashs évitables viennent des mises à jour manuelles de mods ou d’une différence entre la version client et la version serveur.
