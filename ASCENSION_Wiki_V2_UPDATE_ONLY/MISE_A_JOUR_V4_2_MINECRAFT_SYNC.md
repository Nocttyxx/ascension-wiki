# ASCENSION Wiki V4.2 — Minecraft Sync

La page « Mon aventure » accepte désormais directement le JSON produit par :

```text
/ascensionwikiexport
```

L'import Minecraft fonctionne en **fusion** :

- les données détectées en jeu sont ajoutées ;
- les cases déjà cochées dans le wiki restent présentes ;
- les objectifs purement manuels ne sont pas supprimés ;
- la progression personnelle et celle de la compagnie restent séparées.

Le correctif KubeJS doit être installé dans le modpack pour créer le fichier.
