# ASCENSION Wiki V4.0 Alpha — Mon aventure

Cette mise à jour ajoute un tableau de bord interactif personnel.

## Fonctions

- choix parmi les quatre classes ;
- nom de compagnie facultatif ;
- suivi des dix chapitres ;
- quatre preuves automatiques ;
- cinq expéditions majeures ;
- huit rencontres de boss ;
- Six Épreuves ;
- choix final protégé par un mode spoilers ;
- sauvegarde automatique avec `localStorage` ;
- export et import d’un fichier JSON ;
- réinitialisation contrôlée ;
- calcul de progression et prochain objectif dynamique.

## Confidentialité

La progression reste dans le navigateur du joueur. Aucun compte et aucun
serveur ne sont nécessaires.

## Installation

Copiez le contenu de l’archive `UPDATE_ONLY` dans le dossier actuellement
publié par GitHub Pages, en conservant l’arborescence, puis remplacez les
fichiers existants.
