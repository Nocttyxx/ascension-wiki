# Publication du wiki

Le dossier `PUBLIC_WIKI` est un site statique prêt à être hébergé.

1. Remplacer les éléments marqués « À compléter ».
2. Ajouter le logo officiel dans `assets/`.
3. Envoyer tout le contenu de `PUBLIC_WIKI` sur un hébergement statique.
4. Configurer `index.html` comme page d’accueil.
5. Tester les liens sur ordinateur et téléphone.

Le site n’a besoin ni de base de données ni de serveur applicatif.
